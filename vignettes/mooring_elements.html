Dan Kelley

2021-05-18

  • 0.1 Float characteristics
  • 0.2 Wire characteristics

The data stored in data(mooringElements) were extracted from Dewey’s Matlab matlab file named mdcodes.mat, which were apparently extracted from more primary sources. Neither the provenance and accuracy of individual items was checked before inclusion here. Note that Dewey reports dimensions in centimetres, but these are converted to metres here.

The column names are not the same as in Dewey’s Matlab file, although the meaning of the final column is unknown, so it is just called code here.
